出版社: 中国友谊出版公司

副标题: 寻找自我的旅程

译者: 王岑卉

出版年: 2019-8-1

页数: 272

定价: 49.8

装帧: 平装
丛书: 克里斯多福•孟系列作品
ISBN: 9787505747210

简介：他告诉我们如何穿透自我障碍，用爱酿造幸福秘方——没有人能年复一年地活在火辣、热情、浪漫的亲密关系之中，但我们能在亲密关系的旅途中，学习面对自己最好以及最糟的特质，学习接受和放手，最终找到通往爱和幸福的桥梁。

本书分为四个阶段

这四个阶段是打造伴侣关系的基石：绚丽 幻灭 内省 启示

第一阶段 绚丽

1.理想：亲密关系的痛苦之源

很少有东西能像理想的幻象一样，拒绝生命的馈赠

拿实际伴侣和幻想中的理想做比较，可能会感觉不满。当理想的幻象变得比实际拥有的更重要的时候，内心会不满足，无法接受现实。

当只关注理想，就会变得盲目，看不到当下给予的馈赠和教导。

理想的幻象影响到伴侣关系的注意点

  1. 希望对方符合理想伴侣形象
  2. 因为无法满足自我的想象而疏远对方
  3. 寻找更优秀的人
  4. 对理想伴侣的想象，对现有的关系是好是坏
  5. 从理想伴侣和现实伴侣身上找出一点进行比较，观察自己更看重理想伴侣还是现实伴侣？

指导建议：

以下几方面理想伴侣和现实伴侣的评价

分类 理想伴侣 现实伴侣（评分）
风趣与活泼 100% %
热情与激情 100% %
性爱满意度 100% %
浪漫程度 100% %
力量与魅力 100% %
性感程度 100% %
外表吸引力 100% %
对我的需求的敏感程度 100% %

尊重并欣赏我 100% %

结论：

不用关注自己的幻想，只需要意识到理想导致了忽略伴侣，重新关注对方身上的美好品质就好了。只要这样做，就会对当下拥有的东西感到满足，朝着情绪成年大步迈进。

2.期望：滋生自我怀疑的完美土壤

只有抛开期望，才能获得自由

期望是对伴侣施加的一种情感压力

特点：

  1. 对伴侣的期望越大，内心的疑虑就越多，越觉得自己是不完整的。
  2. 有所期望，就一定会失望。期望越大，失望越大。
  3. 期望会附加不言而喻的条件“**如果你爱我…你就会…**”期望只是对受重视感，归属感，安全感，力量感的需求有关，和真正的爱情毫无关系。
  4. 当伴侣肩负起让你开心的责任的时候，就放弃了内心真正的强大-真正的圆满和无条件的幸福

放下期望，不要期待对方提供任何已经拥有的东西

抱怨不会让对方有所改变，越是抱怨，情况越是难以改观。

6.拒绝：远离痛苦，却进一步孤立自己

不该感到遭到拒绝，除非你在拒绝对方

拒绝过程：

  1. 你带着对受重视和归属感的需求进入一段亲密关系。
  2. 你期望伴侣能满足你的这些需求
  3. 伴侣没能满足你的需求，没发让你感到满意。
  4. 需求没有得到满足，让你觉得很受伤，伤痛开始在心中浮现。你在没意识到的情况下做出了反应，通过压抑或避免关注它来拒绝痛苦。
  5. 你说服自己，你感觉到的拒绝来自你的伴侣（我需要某些东西，伴侣没有提供给我，所有说他拒绝了我）。
  6. 事实上，是你在拒绝你的伴侣。完全是因为你让伴侣承担了“满足我的需求”的责任。

指导建议：

  • 我的伴侣，我需要你帮我感觉到自己很重要。
  • 为了感觉自己有价值，我希望你＿＿＿＿（具体解释你希望他做什么事，说什么话。）。
  • 当你没有满足我的需求时，我感觉自己＿＿＿＿（无足轻重、一无是处、没有人爱、没有人要等）。
  • 我会拒绝那种感觉，拒绝你，同时告诉自己，是你在拒绝我。但其实是我对你合上了心扉。
  • 我可能会继续无意识地这么做。不过，我会尽我所能，努力意识到自己的这种行为模式。因为我真的想爱你，想把你从“满足我的需求”的责任中解放出来。

“拒绝”之外的方式做出回应—-比如说接受。
当你接受伴侣无法满足你的需求时，你就接受了自己，转而凭借内在力量满足自身需求。

8.愤怒：引发对方的负罪感及其恐惧

大多数情况下，愤怒只是为了让别人产生负罪感。

人们通常会用以下三种方式表达愤怒：攻击、退缩和被动攻击。不管你用什么方式，模板都是一样的：

  1. 让被人产生负罪感，从而控制他的行为。
  2. 推开自己内心开始浮现的不适感。
  3. 使自己出于有利位置（比如，正确的一方），从而远离不适感。

所有的不适感和信念息息相关，如果忽视了不适感，那种信念就会继续限制你，你却无法意识到这一点。
当感到烦躁不适的时候，继续跟伴侣保持距离，强化自己的孤独感。直到你觉得直接应对靠愤怒来逃避的东西，一切才会有所转变。

9.父母：亲密关系中的隐形负担

我们看不惯父母的做法，却注定要重蹈他们的覆辙。

审视与父母的关系，探讨它对你目前亲密关系的影响。

父母也许给了你很多值得感恩的东西—-你获得的支持，引导和谷粒，还有从他们的人生经历中汲取的智慧。然而，你们的互动过程中也许存在一些不愉快的时刻，成了你目前亲密关系中的负担。

无论你选择怎样的伴侣，他最终会让你想起自己的父母，这几乎是不可避免的。伴侣也许会抛给你某个延伸，或者说出某些伤人的话，跟父母对你的方式一模一样。或许，你会发现自己的做法正是父母让你看不惯的做法。如果上述任何一点，导致你目前的亲密关系出现问题，就标明你过去对父母的不满正在渐渐浮现。如果你无意识地背负起这些不满，而不是从父母的“错误”中汲取经验，就会重复父母的做法。

只要你能看得出伴侣对待你的做法，沟通方式，态度反应了你对待自己的方式，就算百分比很低，你的意识也会有所提升。你对伴侣的过激反应会越来越少，还会将无意识地互动变成有意识地沟通。

9.选择爱：亲密关系的转折点

选择爱的时候，其他东西都不重要了。

亲密关系是为你提供更多满足的体验，所以你完全能让双方的关系和激情永葆活力。情感是浪漫的发动机，源于爱与被爱的需求。总有一天，情感会萎缩，浪漫会消失，激情会消退。此时你就会开始质疑，自己所处的这段关系到底对不对，除了关于爱的浪漫，幻想和伤感哲学外，是不是还有更有意思的东西值得去追寻。如果你的情况正是这样，这就是一个转折点，你有可能从此进入人生的新阶段。在这个新阶段中，您直接感受到无条件的爱和真正的幸福。

你有机会做出选择。你找到的东西也许会比情感联系或依附更有意义。

指导建议：

全心全意地关注伴侣，而不是关注目前的状况或当下的问题。想象你的伴侣就站在你面前。全心全意地欣赏他。向他敞开心扉，想象对方纯净的本质。意识到或看到那种喜乐、平和、充满爱意的本质。重视你的伴侣，胜过希望他满足你的需求。你如果能让自己做到这一点，即使是在想象中做到这一点，就能向前迈进，进入请关系中更深的层次。

你做出的选择觉得你会从伴侣身上看到什么，坚持选择爱，选择真正的爱。

10.转型危机：重温过去错误的绝佳机会

想要做煎蛋，先要把蛋敲开。

转型危机指的是两个人闹冲突陷入僵局甚至闹分居的时刻。

长期受压制的感觉及其强烈，让亲密关系有一拍两散的危险。同时表明巨大的转变正在发送。

不幸的是，这种伤痛往往意味着亲密关系中的冲突或者矛盾的根源。反应过激会失去真正值得重视的人。

太过消极逃避这种情绪爆发，也可能失去这场转型危机带来的宝贵成长机会。只要保持镇定，就会看到，在情绪爆发后，

释然感会油然而生。如今，你忠于可以放下背负那么久的重担了。

指导建议：
如果危机来临，保持内心镇定，尽可能持续关注自己的情绪和感受。努力避免反应情绪过激，也不要逃避亲密关系中发生的事情。

认真体会旧伤的痛楚，用舌头顶住上颚，这样就不会不假思索想什么说什么。

尽可能为冲突带去评价，就像伤口活淤肿敷药一样。

想象自己是一束光，照耀进茫茫黑夜中，投向自己的伴侣，照亮对方的痛苦。继续好好活下去，不被情绪爆发害的全身麻痹。

平静回应伴侣，伤口会逐渐小时，取而代之是美好的馈赠，关系也会更加亲密，爱意融融。

观察自己过去错失的机会，生命给予的馈赠却被自己忽略了。馈赠怎样丰富人生和亲密关系？

补充完整句子。

我错失了体验到＿＿＿＿的机会。

当时，生命给予了我馈赠，能丰富我的人生和亲密关系，那份馈赠是＿＿＿＿。

产开心扉关注会发生生命事情。见证自己成诚意的收下馈赠，抓住机会。

敞开心扉接收给予和馈赠，想象这么做以后会发生什么事情。通过这场危机，亲密关系中会出现什么新领域。

让直觉回答你提出的问题，将答案融入你的内心。

11.独立游戏：持续否认彼此的需求

当你拒绝承认自己的需求时，就是在拒绝自己的人性。

“依赖”和“独立”针对的是我们的需求，尤其是对受重视和归属感的需求。

为了真正走近彼此，最好放弃“独立”或者“依赖”的立场，在本性上和对方建立联系。

独立游戏：

独立痛楚被视为凡事靠自己，不依赖外人、外物。如果你在你们的关系中是“独立”的一方，那么你看上去似乎对伴侣要求不多。你看起来强大而自立。有时候甚至像在照料伴侣。似乎需要什么都能自己搞定，很少展现出自己有迫切需求。

不管你是独立的一方，还是依赖的一方，你们双方：

  1. 进入这段关系都是因为对受重视和归属感的需求。
  2. 无意识地否认这些需求，独立的一方否认得更积极。
  3. 相信独立是力量的象征，但事实上它不过是一种否认的状态，目的是掩饰自己有需求，有脆弱的一面。

独立源于否认自己的需求，保持特立独行，就是持续否认自己的人性，隐藏人性背后的本质。

指导建议：

​ 弄清楚自己扮演的角色。

依赖 独立
充满期待 装作毫无期待
抱怨 从不抱怨
看似比较软弱 看似比较强大，能够自给自足
一心只想着伴侣 看似专注与“其他事物”
视图博得伴侣的关注 注意力没放在伴侣身上
看似“深爱着”伴侣 会说“我爱你”，但看起来不像在深爱着对方
更注重亲密关系 更关注外部世界
比较“忠诚” 比较薄情

独立的角色并不真实，只是在否认自身去修。为了认清真实的自己，就要直面人性的脆弱，透过幻象看清灵性的自我。放弃独立，走近伴侣，就是朝自我迈进。

扮演独立角色，走近伴侣的步骤：

  1. 认识到伴侣的依赖性反应了你自己的依赖性。
  2. 问问自己：“我的伴侣通过口头活支持语言表达了什么需求？”（这些需求通常是受重视、归属感、安全感或者力量感）。
  3. 问问自己：“如果我现在处于伴侣的位置,会希望对方怎么对待我。怎么回应我的需求？”
  4. 遵从内心直觉的指引。请记住，你是怎么对待伴侣的，就是怎么对待自己的。

12.理性谎言：切断深层联系的防御机制

理性解释只会把你带进死胡同。

在亲密关系的某个方面，看不到伴侣建立深层次联系的可能性。这个话题表明，阻止你走近伴侣的可能是你的恐惧。只是伪装成了理性思维而已。理性思维很有用，但当你害怕应对心声催促你去做的事时，理性思维也会被用作防御机制。为了避免直面自己的恐惧，你会编出一大套理性解释，证明自己没做某件事是有道理的—–但这些“理性谎言”最终可能会导致更大的问题。人拥有将某种行为合理化的能力（在你心目中，自己永远是好人），免得自己产生了不适感。然而，不适感是很有必要的，因为它会告诉你，现在是成长的好机会——要是不成长，就会停滞不前。

指导建议：

将下列句子补充完整。

  1. 我和伴侣没有应对的问题是＿＿＿＿（例如,性生活，沟通，亲密感、金钱、双方父母、孩子、工作等）
  2. 对于为什么没有应对这些问题，我给出的理性解释是＿＿＿＿（请描述你和你的伴侣讨论这个问题时用的逻辑论证。例如，他的反应过激了，讨论也没有用，没办法摆脱这种困境，又不是什么大问题，反正这件事我们以后还会再吵的，等等）
  3. 事实上，我避免处理这个问题，是因为它让我感到＿＿＿＿（例如：一无是处、负罪感、不安、气馁、绝望、无助等）

总结：

将某种行为合理化是为了什么，这么做能让你离伴侣更近，还是说智慧让你们越来越远？

理性思维受到了防御机制控制。事实上，你完全可以摆脱控制，转而为真心服务，一旦你做到了这一点，“理性谎言”就会彻底中介，体恤对方的理由就会浮现。即使你还没有准备号走近伴侣，跟他讨论问题或表达自己的感受，你仍然变得对自己更坦诚了。

如果你乐意的话，也可以意识到自己脆弱的一面，并将它们视为能量，而不是视为威胁，这有助于提升你的情绪成熟度。

（当对方情绪不佳，应该听对方说，体会对方的情绪，而不是讲道理）

13.替罪羊：向前一步，向爱靠近

别把对伴侣的看法当做是合上心扉的借口

你会在亲密关系中将伴侣视为“恶人”。你也许会以伴侣的做法或态度为借口，解释自己为什么会不开心，为什么没法更坦诚地去爱。

这背后是你对成长的恐惧。最重要的是要记住，人类最大的恐惧是对爱的恐惧。人类对死亡的恐惧都比不上对爱的恐惧，

因为无条件的爱需要放弃控制、失去自我。就算是失去生命，还有希望在另一个世界保持自我，但如果完全臣服于爱，我们就会出于本能地认定，自己将彻底失去自我。

​ 向伴侣迈出一步，通常意味着卸下心房、向爱靠近。你无意识的防御机制会自动寻找方法逃避爱，也就意味着逃避你的伴侣。因此，在没有意识到自身动机的情况下，你会视图以伴侣的做法、态度或外表为借口，解释你为什么无法在请关系中更进一步。在拒绝伴侣的同时，你可以安全地拒绝爱。

意识到对伴侣的种种解释都只是接口，为了掩饰自己的恐惧。害怕去爱本身并没有错——这是人性，然而，硬是要伴侣为你的人性负责，则会产生副作用。

指导建议：

​ 为了不拿伴侣当做替罪羊，完成以下句子

  • 我的伴侣，我一直在拿你的＿＿＿＿（描述做法、态度或外表）做借口，不愿带着爱意和接纳走近你。

  • 我害怕，如果我走近你，就会感到＿＿＿＿（例如：不安全、没价值、卑微、无力、气馁等）。

  • 此时此刻，我不会拿你当替罪羊，而会＿＿＿＿

将那种感受视为五彩斑斓的能量形态，越是将他视为一种能量，越能深入它的核心，看清自己的本质，发现那些平和、喜乐与爱的力量正等着迎接你的到来。

第三阶段：反省

审视亲密关系的内省阶段，以及其为你提供的摆脱关系僵局、情绪融合和牺牲者角色的机会。

关键在与接收自己人性脆弱的一面，超越它造成的限制，找到自己快乐、平和、充满爱意的本性。

1.倦怠：释放你的激情

倦怠掩饰了对自身创意的恐惧

倦怠感是在压抑某种富于激情的能量。如果把这些能量释放出来，激情与创意的浪潮就会将倦怠感一扫而光。

倦怠感使你处于依赖状态，等待外人，外物为自己提供出路，摆脱倦怠感有98%在于第一步——促使自己付出，或是让自己想要付出。

指导建议：

​ 唤醒你内心热情如火、富于创意的天才，就必须向你的直觉和想象力求助。这两个要素相结合，将为你打开充满无限可能的大门，但请记住，你的创意冲动常常会促使你迈向不敢走的方向，这个时候，你也许会意识到，倦怠感其实是在表达你对自身创意的恐惧。

选择1. 列出想做却没有做的活动（遗愿清单）在清单上分写下你希望和伴侣一起进行的活动或者消遣。它们需要你充分发挥创意，展示你冒险的一面。列好十大清单后，去找伴侣，让对方选择他准备好去试的那一个。等你选定了一项活动，就一步步克服内心的恐惧，把它变成让你兴奋的源泉。

就算伴侣不和你一起做，也可以独自进行。如果动力院子你的内心，就追随你的灵感来源，这将丰富你和伴侣的关系，加深你们之间的联系。

选择2. 尝试情感冒险，打破固有模式，用全新的方式向伴侣表达爱意。这将充分展示你对伴侣的欣赏，同时展现出你的风趣、有创意的一面。采取这样充满创意的步骤，事先不可能知道能否得偿所愿。这会让你摆脱倦怠感，对伴侣充满激情。如果你对这么做的结果不报任何期望，而且本意是好的，那么倦怠感很快就会消失无踪。

2.投射：学会接纳自己的侧面

你越是走近伴侣，就越能看清自己

所有的感知，都是你内心动态和过程的投射。因此，无论从伴侣身上看到了什么，那都是你自己的一部分。你可能并不赞同这个说法。你会看着伴侣的做法，一口咬定：“我才不会这么做呢”

当你将伴侣和其他人视为自己的投射时，就会提升意识，走向成熟。通过接纳这种投射，你可以收回自己的一部分，一旦两者结合，投射就会变成宝贵的馈赠。

指导建议

回答下列问题

  1. 描述你不喜欢伴侣的某种行为＿＿＿。
  2. 你认为这种行为背后的态度是什么？（例如：批评、控制、操纵、粘人、不支持、争强好胜、不知感恩、漠不关心、不在乎、不信任、怀疑、妄加评判等）＿＿＿＿
  3. 你是否发现自己也有统一的态度，只是表达方式不一样？
  4. 这种态度受到哪种基础情绪的影响？（例如：一无是处、对受重视和归属感的需求、不安全感、被遗弃感、昔日的心碎、焦虑、恐惧等）＿＿＿＿
  5. 你是否发现这种情绪也在影响自己？
  6. 重复上述练习。

感受想象伴侣的态度和情绪完全融入你的内心，在融合的过程中，充分地接纳他们。

通过接纳你投射到伴侣身上的那些侧面，对方可以卸下重担，不再受你的想法攻击，你们之间的隔阂也会小时。与自己的投射合二为一时，你会意识到这是一份馈赠，他会进一步揭示你的本质。

3.道歉：与原谅无关

任何人都可以说对不起，但只有成年人才能真正的道歉。

真正有效的道歉能帮你意识到：

  1. 体会到负罪感
  2. 你背负的伤痛或需求才是恶行的根源。

指导建议：

补充完整。

  1. 我最近一次伤害伴侣的时候，做了（没做、说的是）＿＿＿＿。
  2. 互相自己当时的做法，我感到＿＿＿＿（例如，负罪感、伤心、羞愧等。）
  3. 我在那样做之前就（有意识或无意识）体会到的不适感是＿＿＿＿（例如：一无是处、被人抛弃、失望、失败、微不足道、没有人要、无力、无助等）

不期望对方的原谅，也不指望他做出热情的回应。直面对方的反应，也直面自己内心最脆弱的东西，而不要竖起心防。

道歉与原谅无关，只是帮助自己接收大半辈子都在压抑回避的东西，那才是通往真正自由的大门。

注意事项

“无心的伤害”常常被人忽略，把直面内心深处的负罪感放在第一位，接受并意识到自己的感受，最终就能体会到自己的无辜，而不用费尽力气试图证明这一点

4.承诺：给予对方无条件的爱

许下承诺之前，什么事都不会发生

承诺与时间无关，只与当下有关，

对自己的情绪和内在成长许下承诺。你对自己许下承诺之后，伴侣关系就能帮你在无条件的爱和真正的幸福中获得成长。如果没有对自身成长许下承诺，就会从有条件的爱和有条件的幸福出发，看待自己与伴侣的关系。

当你认可伴侣的做法的时候，就会感到幸福并深爱对方，当不认同的时候，就会感到不幸福，不爱对方，或者说没有对方采取你认可的做法时那么爱他。

“有条件的爱”是指需要满足一定条件才能体验到你渴望的评价，喜乐。

有条件的爱是暂时的——你对伴侣寄予厚望，希望对方以某种方式说话、做事。无条件的爱则能让你体验到爱的平静、喜乐——不管你周围的情况如何，也不管你身边人的行为方式如何。无条件的爱关注亲密关系的真正目的，而有条件的爱值关注满足你的个人需求。

拿这两种选择做个比较，不断地提升你的意识，认识自己充满爱意，喜乐、平和的本性，这才是亲密关系中唯一值得许下承诺的。要向亲密关系的真正目的许下承诺，而不是向“无期徒刑”许下承诺。不过，这个承诺可能会持续终生。

指导建议：

在亲密关系中提升意识，了解自己的本质，

  1. 想象有2个一模一样的自己，一个感觉自己很特别，希望周围的人都觉得自己特别。一个想摆脱这种需求，想从内心找到幸福，体验真正无条件的爱。

  2. 想象2个自己都站在面前，

  3. 关注想靠外物满足自身需求的你，观察他脸上的标签。

  4. 然后关注香葱内在体验无条件的爱和真正的幸福，观察表情。

  5. 来回看两个自己的投影，观察自己的感受。

  6. 想象两者合二为一是什么感受，看到这样的“你”又有什么感觉？

  7. 这个人会选择在亲密关系中许下什么样的承诺？如果你愿意的话，融入心中。

    在内心深处，你一直在向自己许下承诺。

6.痛苦时给予爱：消除痛苦最直接的方式

只要心脏还在跳动，你就能将爱给予对方。

7.嫉妒：邀请接纳和平静进入内心

嫉妒是一种痛苦，源于对特别感的渴求。

寻找摆脱困境的方法，否认自己在吃醋就是在疏远伴侣。如果靠近伴侣，就会意识到自己有多依附对方，多希望对方让你成为世界上最特别的人。当你觉得伴侣给某人的关注或者爱意比给你的多的时候，你就会倍感痛苦，受尽折磨。难以战胜的嫉妒会激起好胜，仇恨，愤怒和需要。

指导建议：

  1. 回想引起你嫉妒的事的细节
  2. 想象你体内充满嫉妒的感觉，如果嫉妒看似比你的身体还要大，那就想象你的身体不断扩展，直到把嫉妒全都包容在内。
  3. 把跟那件事有关的图景全部抛在一边，把注意力全放在身体的感受上，关注自己的感受的核心在哪里，把意识引入那个核心。镇定、平静地观察它的能量，敞开心扉，接收自己的人类体验。保持敞开心扉的方法之一。，就是想想象你真正在乎的人。在脑海中映出那个人的木有，会有助于你保持敞开心扉。
  4. 当你要求接纳和评价进入嫉妒之心时，嫉妒就会渐渐消融，显露它背后隐藏的东西。

将嫉妒转换为爱（不管是奉献、赞赏、想象还是选择爱的愿望）都能将嫉妒放松对你的控制，使你更接近自己的本质。

一切都会过去

  1. 证明对方是错的，
  2. 或者这些问题让你感到不舒服

无论问题是什么，不愿意接受它的存在，只会使问题复杂化，让你和伴侣产生隔阂，使你远离内心的平静。

指导建议：
摆脱问题的第一步，接受问题的本来面目，既然抵抗是一种基于恐惧的反应。那么抵制问题只会夸大它的威胁，让你觉得问题太大，而自己太弱小。臣服能让你摆脱受害者认知，从强者的角度看待问题。

臣服能帮自己后退一步，客观地观察这些体验，提醒自己“这种感觉会过去的”

抵抗或反应过激不会带来你想要的结果。只有接受问题的本来面目，你才能将它视为漫漫人生路中的一个阶段，学会在平静、理解等方面得到成长。一旦你敞开心扉，接受处境的本来面目，提醒自己“一切都会过去的”，它就会变成一股强大的助力，使你免受当前状况带来的痛苦，消除你和伴侣之间的隔阂。

  1. 一旦发现愤怒或其他情绪反应，就意识转为从外向内看。最好后退几步
  2. 想象愤怒的情绪，防备感，我永远是对的，念头，当对自己说“一切都会过去的”想象盔甲慢慢消融。
  3. 继续观察情况，关注自己想要跳到外面，为冲突而纠结的冲动。
  4. 意识到你内心的感受，你的心在说“比起这么苦苦挣扎，我更喜欢平静”。
  5. 当你观察外界的情况时，也要注意自己脆弱的一面，以及内心的感受和伤痛。
  6. 你的伴侣可能会将你的臣服视为退缩。然而，如果你发自内心地表达感受，向可能出现的任何情况臣服，平静与联结最终会占上风。

4.满足：当给予和接受达到平衡

让爱所规划的目的自然显现

在亲密关系中，是否有感受过满足？回想当时在做什么，说什么，分享什么，让你意识到满足？

满足是动态的过程，是给予和接受的平衡，双方没有任何区别，才会感到满足。

当摆脱自己有意无意强加给伴侣的需求和期望，就会发现这种“给予”和“接受”的循环才是你最想体验的。

只有百分百的“给予”和百分百的“接受”的过程，才会感受到无上的满足。

指导建议：

按照以下步骤操作：

  1. 想象伴侣站在面前。

  2. 想象你在等她展示对你的爱意、赞赏和珍视。想象伴侣以客观、中立的姿态站在哪儿，同时体察自己的期望和需求。至少三十秒。

  3. 关注身体和情绪上的感受。你会说自己体验到了百分之几的满足？ ____%

  4. 迈出一大步：闭上双眼，摆出客观、中立的表情，想象你的伴侣站在你面前。

    这一次赞赏你的伴侣，从内心深处感觉到你对伴侣真实模样的感激、爱意和经纬，感受从中迸发的能量，意识到他是一种充满爱意、智慧和喜乐的灵性存在，只是寄居在人类的身体里，聚精会神地观察自己内心用处的赞赏的能量。至少30秒。

  5. 关注你此时此刻身体和情绪上的感受。和之前相比是否更加满足，将注意力放在“给予”上，而不考虑回报，就会感到满足。关键在于，是“给予”让你满足的。视图从别人哪里得到什么，智慧让你深感空虚，难以满足。

总结：给予和接受是完全相同的体验。如果你子啊等待伴侣给予你什么，实际上就是对伴侣抱以期望。因此，你收到的只会是你的期望。如果你给予对方爱，就会从对方那里收到爱，这就是得到满足的秘诀。

6.优先考虑：什么会排在人生第一位

当你优先考虑真相的时候，爱与幸福就会随之而来。

让你审视人生中的优先考虑。你的体验取决于你的优先考虑，你的优先考虑则取决于你认为需要什么东西才能得到幸福、成功、充实和安全。当你认为外物能带给自己充实的内心体验时，一切都会分崩离析。如果你认为自己必须达到某种财务状况才能感觉良好，那么无论你的伴侣有多优秀，双方的关系都没法让你满意。如果你认为伴侣必须以某种方式满足你，控制对方就会成为你的优先考虑，而更有可能令人满足的爱与接纳则会变成次要考虑。

如果你与伴侣长期发生矛盾，而你的优先考虑是控制对方，那么幸福永远只能排第二位，你的体验与你的优先考虑总是息息相关。那么，你在当前关系中的体验表明你的优先考虑是什么？

事实证明，以下这种排序比较理性。真相排在第一位。问题排在最后。

  1. 真相（爱，真正的幸福等）。
  2. 亲密关系；
  3. 问题。

选择优先考虑真相，就选择了真正重要的东西，而不是个人的需求、恐惧、伤痛或负罪感。你的头脑会专注于思考怎么做对双方都有好处。内心的关注焦点会影响你在亲密关系中的行为和回应，然后，源于爱意的回应会影响你处理问题的方式，最终得出和平的解决方案。
