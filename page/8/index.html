1393.历史丨《经略幽燕》：“高梁河车神”打不赢辽朝、收不回幽燕的战略分析

发表于 2021-07-17 分类于 听书笔记

“高梁河车神”打不赢辽朝、收不回幽燕的战略分析

精华笔记

一、 宋辽两国的对峙是怎么开始的

1.辽朝“维持现状”的大战略

1.1 辽朝积极经略中原。石敬瑭割让幽云十六州，中原门户大开。

1.2 辽朝经略中原失败，转向维持现状的大战略，以长期统治幽燕为核心目标。

2.宋朝“维持现状”的大战略

2.1 宋初大战略有和平与统一双重目标，两个目标存在矛盾。

2.2 后周王朴提出“先南后北”的统一战略，协调两个目标的矛盾，并被宋太祖继承。

2.3 先南后北战略提出，先统一南方，再以南方经济支持北方的战争。这样宋朝就要求与辽朝暂时维持现状。

3.雄州和议与宋辽对峙的形成

3.1 宋辽双方倾向于维持现状，双方都保持克制。表现：宋辽在北汉境内大打出手，但在直接接壤的河北地区却保持克制。

3.2 970年，宋辽定州之战，辽朝首次在河北出动6万大军，宋辽冲突可能进一步升级。

3.3 宋辽双方为避免冲突升级，于974年缔结雄州和议，宋辽对峙局面获得双方确认。

4.雄州和议的不稳定性

4.1 宋朝未放弃收复幽燕，这与辽朝保有幽燕的大战略存在根本冲突。

4.2 宋太祖制定了文统与武统两个收复幽燕的战略，概括而言是：外交手段优先，但绝不放弃使用武力。

二、 五代的军事信念如何导致宋军在高梁河战役（第一次经略幽燕）中惨败？

1.五代军事信念，导致宋太宗实施武统

1.1 军事信念：是一种在过去的实战中积累起来的战斗经验，人们往往认定这种经验是正确的。

1.2 五代军事信念的特点：战略奇袭，皇帝亲征。

1.3 继位合法性受到质疑的宋太宗，急需按五代传统，通过亲征幽燕树立威信，979年发动第一次经略幽燕的战役。

1.4 宋太宗军事经验不足，直接模仿五代战略奇袭。

2.高梁河战役第一阶段：宋军突袭，势如破竹

2.1 宋军突袭的军事效果：辽朝猝不及防，宋军势如破竹（个案：急袭岐沟关，渡桑干河）。

2.2 宋军围攻幽州的军事部署：四面围攻，曹翰率预备队待命。

2.3 辽援军兵力分散，只能退驻幽州外围。

2.4 宋军一度登上幽州城头，幽州及周边的官民归附，辽在幽燕的统治几乎瓦解。

2.5 战略奇袭有效的原因：敌人体量有限，奇袭成为致命一击。

3.高梁河战役第二阶段：辽军援救，宋军溃败

3.1 战略奇袭的局限性：辽朝体量大，削弱了奇袭效果，有机会重夺战役主导权。

3.2 宋太宗忽略山路防御，辽军走山路绕到宋军后方。

3.3 宋太宗将曹翰预备队用于攻城，导致宋军无法应对辽军在后方的突袭。

3.4 宋军在高梁河被辽军击溃，宋太宗重伤弃军而逃。

3.5 宋人不再迷信战略奇袭和皇帝亲征，高梁河战役成为最后一场“五代式战争”。

4.高梁河战役在宋军指挥系统中的后遗症

4.1 皇帝不再上前线，但仍遥控前线，宋军指挥系统出现问题。

4.2 满城会战，宋太宗通过阵图遥控前方将领，险些导致宋军战败。

4.3 瓦桥关会战，宋军不设前线总司令，前线各部队各自为战，无法配合，宋军虽然击退辽军进攻，但伤亡惨重。

三、是什么导致了宋军在雍熙北伐（第二次经略幽燕）中铩羽而归？

1.宋朝转而实行内政为上的大战略

1.1 内政为上大战略：通过优先发展内政来增强综合国力，然后以国力上的优势来决定何时再次北伐。

1.2 实际情况相反：辽朝进一步解除后顾之忧，而宋太宗只忙着巩固个人权力，局面对宋朝更加不利。

2.各层次战略的脱节，导致雍熙北伐溃败

2.1 986年，宋太宗发动雍熙北伐。

2.2 大战略和战役规划脱节：宋朝国力尚不足以压倒辽朝，发动北伐违背了内政为上大战略。

2.3 战役规划与战术行动能力的脱节：计划分东、中、西三路大军北伐，采用声东击西，最终三路大军围攻幽州；实际上东路军却屡屡失误，最终在岐沟关被辽军击溃。

四、 宋军是如何转入战略防御，放弃收复失地打算的？**

1.宋太宗第三次经略幽燕的部署

1.1 986年冬，宋太宗为第三次经略幽燕做军事部署。

1.2 部署：李继隆渡海攻后方，刘廷让主力北伐，田重进策应。

1.3 宋辽形势已发生变化，辽军转守为攻，宋军却未转入防守。

2.宋军在君子馆溃败

2.1 弹性防御：在应对敌人入侵时，只要将敌军击败就可以达到保卫国土的目的。

2.2 986年冬，辽军征宋，宋将刘廷让的最佳方案是弹性防御。

2.3 弹性防御的重要条件是各部队间的有效协调，而缺乏前线总司令的宋军不具备这一条件。刘廷让被迫选择正面迎敌。

2.4 以步兵为主力的刘廷让选择了正确的战术部署：弓弩、拐子马、纵深阵列。

2.5 刘廷让与辽军战于君子馆，因气温骤降，宋军无法拉开弓弩，技术优势丧失。

2.6 天气恶化动摇了协同作战，不受刘廷让节制的李继隆放弃支援。

2.7 最终刘廷让在君子馆战役中大败。

3.宋朝转入战略防守

3.1 宋军高层转向慎重和避战，宋军防御战取得胜利。

3.2 个案：张齐贤取得土嶝寨之捷。

3.3 形势恶化，宋朝放弃北伐，转攻为守。

3.4 辽朝需要巩固在幽燕的统治，没有进一步征宋。

3.5 宋辽回到维持现状的对峙局面。1005年，两国缔结澶渊之盟，进入和平对峙新时期。

1392.经管丨《稀缺》：穷不是一种状态，而是一种心态，解决了心态，才能摆脱贫穷的状态

发表于 2021-07-16 分类于 听书笔记

> 穷不是一种状态，而是一种心态，解决了心态，才能摆脱贫穷的状态

精华笔记

贫穷与忙碌听上去毫无关系，如果你深入探究，就能发现二者之间有一个共同点，那就是“拥有”少于“需要”的感觉。有意思的是，就算给忙碌的人更多的时间，给穷人更多的钱，他们仍会处于忙碌和贫穷的状态中。

这种现象就是“稀缺”。这里的稀缺，不仅是指缺少时间和金钱这种实质上的资源的匮乏，还指一种心态。当你缺少时间或者金钱时，注意力全部盯着眼下的某物或者某事的状态。

一、为什么说稀缺是让贫穷的人更穷，忙碌的人更忙的根本原因呢？

第一个原因是稀缺导致管窥效应。

什么是管窥效应呢？当我们透过一个管子去看东西，只能看到管子空洞范围内的少量物体，看不到管子之外的其他东西。你急于完成报告，由于临近截止日期，你只专注做报告，不管其他事情，这就叫做管窥效应。

管窥效应能让人更专注、高效地做事情。这份专注就是稀缺带来的好处。不过，相比于好处，管窥效应带来的坏处更大。

作者认为，管窥效应会从两个方面影响人们的决策：

第一个方面是，

管窥效应让人只一门心思专注手头上的事情，却可能忽略其他同样重要的事情。

美国曾有一份针对消防员死因的统计数据，在出火警时，有20%-25%的消防员不是葬身于火场，而是死于出火警途中的交通事故。在这些交通事故中，又有79%的消防员是死于没有系安全带。根据调查，这些消防员在日常生活中没有不系安全带的行为，他们甚至还会提醒家人要系好安全带。这就是管窥效应带来的后果。在接到火警报警后，时间非常紧急，消防员们要在第一时间赶到火场，并在极短时间内制定出灭火的策略。因此，系安全带成了管子以外的事情，就容易被他们忽视。

管窥效应对人们决策的第二个影响是，人们会做出同时处理多项任务的决定。

比如说，当你特别忙的时候，你会一边听电话会议，一边回邮件，这样同时做几件事情，似乎时间也变多了，这正是人们认为的管窥效应之内获得的收益。但是，同时处理多件事情可能会出现的问题，则处在管窥效应之外。比如导致电话会议漏掉重要信息，或者邮件写错收件人，等等。这些失误则不在你的考虑之中。

稀缺导致穷人更穷，忙碌的人更忙的第二个原因是，没有余闲导致的借用效应。

什么是余闲呢？余闲，就是人在拥有很大空间、不存在稀缺心态时的产物。稀缺本质上就是没有余闲，满满当当的时间表不能多一件临时任务，各类账单排下来，工资被用得精光，不能承担多一项的额外支出。但生活总是充满了意外，当临时任务、额外支出出现时，人们无法腾挪，只好去借用。

借用，虽然解决了眼下的燃眉之急，但它却增加了未来的成本，因为你早晚要归还的。借钱要支付利息；借用时间，也许会因为没有做这件事情付出成本，或损失这件事情可能带来的收益，又或者在将来要用更多的时间来完成。

作者还发现，越是穷人越喜欢借高利贷，越是忙的人越喜欢拖延时间。

稀缺让穷人更穷，忙碌的人更忙的第三个原因，也是最重要的原因——带宽不足。

什么是带宽呢？

作者将一个人的精力、智力称之为带宽。带宽对人很重要，因为人们需要充沛的精力去读书、思考，需要更多的自我控制力去专注地工作，需要更多的洞察力去为自己的将来做出更长远的规划。而稀缺会降低所有这些带宽的容量，比如说，如果手头上的重要工作没做完，你在做其他事情的时候也会时时想起这个工作。

穷人和忙人的带宽通常更窄，因为他们每天遇到的紧急事态占据了他们大部分的带宽，他们没有精力去关注那些不紧急却很重要的事，比如美国的穷人普遍更加肥胖，不会控制饮食；再比如穷人女性在怀孕后，不懂得合理饮食的重要性，也不做产前检查，因此更容易生下有问题的孩子。而造成这些事实的原因，就是因为穷人已经没有足够的带宽留给重要的事情了。他们每天为生计发愁，哪里还有心思记得这些事情呢？

二、人们如何才能摆脱稀缺，走向富足呢？

作者认为，最主要的措施有三个：第一，将重要的事情推入管子内，这样人们就不会有管窥效应之见；第二，增加余闲，这样就不会总是借用，有能力应对突发的事情；第三，优化带宽，也就是提升自己的认知，做长远的规划。

这些措施具体实施时，又可以分成为国家、组织和个人三个层面。

先来讲国家和社会层面，因为扶贫首先是国家层面的事情，要想减少世界上的贫困，就要先通过国家制定政策来帮助穷人。

那么具体该怎么做呢？是多给穷人发救济金？还是让他们去参加某个复杂的福利项目？作者认为，都不是。而是要通过对稀缺心态的把握，设计出合适的激励政策。

首先，帮助穷人将重要的事情推入管窥效应内。比如在农民刚丰收、手里比较宽裕的时候，政府可以推荐他们购买收成更好的高价好种子。

其次，帮助穷人增加余闲。政府可以尝试向穷人提供能构建起储蓄余闲的金融产品；或者量身定制一种更理想的方案，开发出一款金融产品，将农民在收获时赚得的收入进行储蓄，然后再平摊到每个月，从而有效地将一次性收入转换成月收。

第三，要节约穷人的带宽。给穷人提供的培训课程要简单易懂，容易坚持和完成。

再来看企业和组织层面。

企业和组织主要是要在余闲上加强管理。因为如果一个企业总是满负荷工作，往往并不能创造更多收益，反而会带来更多问题。

第一，工作不要排得太满。就好比道路占用率为70%时，能达到最佳运转状态。如果道路占用率达到了85%，虽然车辆之间还是能留下一点空间，但如果一位司机踩了一下油门后紧跟着又是一脚刹车，那么他后面的每辆车就都要踩刹车。如果后面的车刹车不及时，出现了追尾事故，那交通速度就将大幅下降，甚至让路面陷入瘫痪状态。

第二，企业要设立一个专门的人或者部门管理余闲，确保余闲存在。他的日常工作就是关注明天可能会出现什么样的突发事件，这些突发事件又会如何影响紧张的工作计划，并且及时协调，在问题发生前就将问题解决。

最后来说说个人层面。

作者的建议总结起来可以分为三点：

第一，引入“提醒”机制，把重要的事情拉入到管窥效应中。比如你可以设置一个定时邮件，在发工资的那天提醒自己存一部分钱到专门的储蓄账户。

第二，留有余闲。在金钱上，要学会储蓄，不要透支，消费要量力而行。在时间上，要给自己留有空闲，不要把时间安排得太满。只有留有余闲，才能够从更宏观的层面看待自己的事情，做好长远的人生规划。

第三，节约带宽。一个人的精力是有限的，所以不要在小事情上过于纠结。要把注意力放在真正值得的事情上，集中在可以长期积累的事情。

一个人的精力也是有起伏的，不会一直保持同一个状态，有时候精力充沛，有时候比较疲劳。所以要合理使用带宽，在自己状态最好的时候，做最重要的事情。

稀缺给人造成的负担非常沉重，人们深受贫穷和忙碌的困扰。只有了解了稀缺的内在逻辑，我们才能尽量缓解其负面影响，从被动应付到主动应对。希望通过对《稀缺》这本书的了解，能够让你多一些应对稀缺的方法，变得真正富足和从容。

1391.社科丨《父权制与资本主义》：“男主外、女主内”的分工是合理的吗？

发表于 2021-07-14 分类于 听书笔记

“男主外、女主内”的分工是合理的吗？“全职妈妈”是一个好选择吗？

本书作者上野千鹤子是日本最著名的女性主义学者，现任东京大学人文社会学系教授。著作包括《近代家族的成立与凋零》《父权制与资本主义》《裙子底下的剧场》《厌女》等。

本书分析了安在当代女性身上的两座大山，即资本主义和父权制；这种双重的控制，使女性到今天依然处在屈从的地位中。

第一，资本主义和父权制对于女性的压迫与共谋。

马克思主义认为，在现代的资产阶级社会，女性和无产阶级男性一样，都受到资本主义的统治；而激进女权主义将矛头指向父权制，即家庭之中由年长男性掌握权势的制度。

本书作者上野在内的新一代女性主义者，认为这两边说得都有道理。现代女性既受到资本主义的统治，也受到父权制的压迫；不仅如此，这两者之间还存在着对于女性的共谋，而他们共谋的落脚点，是现代家庭。

今天我们熟悉的这种一夫一妻制的现代家庭形式，附带着一种“男主外、女主内”的性别分工模式，即男性投入公共领域的劳动市场、成为社会生产的劳动力，而女性逐渐被社会劳动市场排除在外，囿于家庭，成为家庭主妇。这种性别分工带来了“市场”与“家庭”的分裂，也让女性深陷从属的地位。

第二，现代家庭通过社会再生产劳动让女性深陷从属的地位。

在现代社会，市场从家庭中获得劳动力，而将无法成为劳动力的人返回给家庭，由女性负责照料。也就是说，女性通过在家庭的劳动，从外部维持整个市场的顺利运转。这些劳动被称为**“社会再生产劳动”，即一系列与生命相关的劳动，包括对人的生育、养育、照料、陪护、等等。**

在父权制和资本主义的双重控制下，几乎所有的女性都在无偿地付出社会再生产劳动，但没有获得任何实质性的回报。这其中起作用的是意识形态的力量。

> 动物是怎样利用物理学在野外生存的？

精华笔记

许多动物都会利用物理规律提高生存效率，并在不同的领域表现出不同的优势，比如在热学、力学、流体、声学、电磁学和光学等领域，都有相应的动物物理学高手，让我们来分别见识一下。

一，热学领域

蚊子吸血时的速度非常快，它们必须在瞬间吸食大量人血，体重瞬间猛增三倍。而如此快速大量吸血却会造成一个意外的麻烦。蚊子是冷血动物，最舒适的体温是25℃左右。但人是恒温动物，体温可达37℃甚至更高，所以蚊子在短时间内大量吸食人血，会造成体温过高，就像是喝多了的醉汉，行动笨拙而迟缓，就此成为其他蚊子的攻击目标。要想避免这种悲剧，蚊子就必须防止身体过热，但它们又不能慢慢吸血，那样只会死得更惨。解决的策略就是从体内再排出一部分血液，顺便把尿也排出去，并通过排出的液体带走大量热能，以此为身体降温。不明真相的人看到这一幕时，还以为蚊子把自己肚子撑破了。

二，力学领域

壁虎可以在天花板上游走自如，就是利用了范德华力的结果。范德华力就像小型的万有引力，无处不在。可是范德华力只能在10纳米的距离内起作用，为此壁虎需要尽可能地将脚趾贴近墙壁，粘贴的面积越大，范德华力就越大。为此壁虎的脚掌进化出了复杂的肉垫，上面覆盖着层层褶皱，褶皱上长满了细细的纤毛，可以最大程度增加与墙壁的接触面积，提高了与墙壁之间的范德华力，足以把自己挂在墙壁上。

与此同时，壁虎的脚趾结构非常灵活，可以一点点弯曲收缩拉离墙壁。当所有脚趾都被拉起来时，它就可以离开墙壁了。整个过程速度非常快，壁虎因此可以抓住前方的昆虫。

三，流体领域

猫的嘴裂较大，不方便直接把嘴插在水里吸水，它们只能用舌头舔水。通过高速摄影机，研究人员拍下了清晰的猫喝水画面，通过慢放镜头发现，它们把舌头卷成了勺状，先用勺子背面在水面沾一下，然后飞速提起，利用内聚力将水拉起来，形成一根细长的连续水柱。在水柱被重力打散之前，猫的下巴迅速闭合，将水包在嘴内。由于每次吸到的水量有限，所以猫必须不断重复这个动作，甚至在一秒钟内舔水三四次，有时要花好几秒才能喝下一口水。如果不是水的内聚力，恐怕它们需要付出更大的努力才能解渴。

四，声学领域

动物要想感知声音信息，就需要敏锐的耳朵。但大象却可以用脚感知额外的声音信息。因为大象体型很大，吼叫声同样也很大，在叫声中还夹杂着一些次声波。当次声波冲击地面时，可以顺着地面继续传播，形成地震波。大象脚上长有厚厚的脂肪组织，其中分布着大量帕氏小体，那是感知轻微压力信号有专用器官。帕氏小体会将地震波信号通过神经传递给大脑，这样大象就用脚听到了远方的次声。

大象在聆听地震波时，往往会抬起一只脚，以便将体重均匀分布在其他三只脚上，导致三只脚承受的压力更大，对地震波更敏感。而且大象腿之间的距离很远，三只脚构成了典型的锐角三角形。大象可以利用三角测量原理，计算地震波传来的方向和距离。

五，电磁领域

花朵暴露在空气中，而空气中往往带有大量负电荷。当花朵携带了大量负电荷时，就会形成一个小小的电场，为蜜蜂提供了一个有效的线索。

蜜蜂的身体总是毛茸茸的，那正是本书名字的由来。细绒毛是探测电场的重要工具，蜜蜂能感受到绒毛的变化。越是靠近花朵，绒毛的静电感应就越明显，蜜蜂的感知就越强烈，那时它们就会知道，自己正在接近一朵花。

当蜜蜂在空中飞行时，绒毛会和空气摩擦而失去电子而带正电。当带正电的蜜蜂拜访带负电的花朵时，就会通过静电作用将花粉吸到身上来，同时中和了花朵的负电荷，进而影响花朵的电场强度。当蜜蜂发现一朵花的电场变弱时，就知道已经被其他蜜蜂采集过了，因此不必在这朵花上浪费时间，大大提高了采集花蜜的效率。

六，光学领域

澳大利亚有一种神奇的射水鱼，它们会悄悄潜伏在湖水中，如果发现猎物，就会出奇不意地从嘴里射出一道水柱，像子弹一样精确地射中猎物，甚至摆脱了光的折射的干扰，几乎百发百中，它们是怎么做到的呢？

原来射水鱼的眼睛与众不同。它们眼底的视网膜特别发达，在关键感区域，仅一平方毫米的范围内，就包含有五万多个感光细胞，比其他感光区域多出了十倍。如此密集的感光细胞，可以收集水面各个方向折射过来的光线，并通过对比消除折射干扰，得出准确的猎物位置。一旦锁定猎物，它们就不会轻易移动眼睛的位置，而是靠调节身体方位寻找最佳发射角度。一旦调节工作完成，它们就会射出一道强大的水柱，将猎物一击致命。

许多动物都在各自擅长的物理领域取得了惊人的成绩。这本书的内容相当丰富，总计介绍了几十种动物物理学技巧，每种技巧之间不存在紧密的逻辑关系，就像是不同专业之间彼此隔阂一样，唯一的共同之处就是全都令人叹为观止。

> 18种语言版本热销全球，辨别不良思维习惯戒掉坏情绪

精华笔记

人们通常认为负面情绪的来源是让人不开心的具体事件，但本书作者安德烈娅·欧文认为，事件背后潜藏的不良思维习惯，才是造成负面情绪的罪魁祸首。

想消除负面情绪，我们要勇敢地直面不良思维习惯，去辨别和处理它们，同时学会敞开心扉去关爱自己，这样才能获得真正的自由和安宁。

一、缺乏自信

1、自我批评

成因：热衷批评的原生家庭和社会文化。

解决方法：列出自我批评和对应的触发事件，在自我批评出现时进行友善的自我对话。

2、冒充者综合征

成因：热衷批评的原生家庭和社会文化。

解决方法：列一份成就清单，写下自己能实现这些目标的原因，肯定自己的成功。

3、灾难恐惧

成因：过去没有得到足够的爱和认可。

解决方法：

①练习对当下的感恩。

②请亲密的朋友告诉你，为什么他们会喜欢你。

二、恐惧失败

1、完美主义

成因：有完美主义习惯的原生家庭，或者有非常优秀的兄弟姐妹。

解决方法：

①比较完美主义给自己的得失。

②允许自己做一些不那么完美的事。

2、过度成就

成因：有过度成就习惯的原生家庭。

解决方法：

①改变对失败的看法。

②将注意力集中在少数几个重要领域去取得成功。

3、拒绝求助

成因：以前向别人求助时遭遇了失败。

解决方法：寻找富有同情心的见证者，体验真正的关爱和支持。

三、压抑情感

1、麻痹负面情绪

成因：负面情绪会让人痛苦、美国文化将表达负面情绪视为“不坚强”。

解决方法：

①重新理解坚强。

②信任自己的情绪。

③用恰当的方式释放负面情绪。

2、取悦他人

成因：父母要求孩子满足自己要求，才会表现出关爱和认可。

1384.社科丨《个体化》：今天的社会中，每个人都想“为自己而活”

发表于 2021-07-06 分类于 听书笔记

> 今天的社会中，每个人都想“为自己而活”

精华笔记

在今天的社会中，一个声音异常响亮，这就是“为自己而活”。每个人都有种强烈的欲望，要以自己的意愿来度过这一生。这种理念的增长，被称为“社会的个体化”，我们现在的社会，也被称为“个体化的社会”。

一、什么叫做个体化，个体化的社会是如何形成的？

个体化是一种转变的趋势，具体来说，是人不断地从各类社会共同体，例如国家、阶级、家庭、族群等脱离出来。在个体化成为社会发展最重要的趋势之前，个体强烈依赖于刚才所说的各类共同体。只有经过了个体化过程，人们从这些共同体中脱离出来，不再嵌入。

个体化社会的形成有三个重要的因素，分别是古典个体主义的思想遗产、劳动力的自由流动和福利国家兴起。

二、生活在个体化社会中，人们的生活发生了什么变化？

第一个后果，是标准化的人生变成了选择性的人生。随着个体化进程的深入，个体必须给自己的身份定位，必须自谋生计，去开创自己的人生，并在与他人的关系中安排自己的人生。例如，家庭从“需求的共同体”变成了“变成一种选择性的亲密关系”。

第二个后果，是风险与责任都被个体化了。在个体化社会中，人们极力地想为自己而活，但是生活的世界却不是以个人的意志为转移的。人生中的风险与危机，有一部分是个体可以预计和控制的，但也很大一部分不是个体可以控制的。进一步地，风险与责任的个体化造成了两种情况。第一种情况，个体必须周而复始地、始终如一地为生活而努力，就好像站上了跑步机，就必然要不停地奔跑。第二种情况，在周而复始的努力中人们也容易堕入了虚空。

三、个体化的社会还有可能培育公共精神吗？

个体是民主体制或共和体制的基础，也是一切政治组织的基本单元。仍然要大力地保护个体自由。不是简单地通过议会和政府来实现民主，而应该在更小的社会单元，例如地方和社区中组织，着眼于公民的日常生活层面，从而建立起自我负责、积极主动的公民社会。

作者提倡一种“合作的个体主义”，每个人都有权为自己而活，但是我们共同生活在一起，因此在社会生活中，必须重新协商共同生活的规则。
