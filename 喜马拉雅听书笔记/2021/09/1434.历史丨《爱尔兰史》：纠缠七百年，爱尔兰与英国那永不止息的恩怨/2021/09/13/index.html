1434.历史丨《爱尔兰史》：纠缠七百年，爱尔兰与英国那永不止息的恩怨

发表于 2021-09-13 分类于 听书笔记

纠缠七百年，爱尔兰与英国那永不止息的恩怨。

精华笔记

一、从盖尔人到爱尔兰人

1、盖尔人的部落邦国

1.1 公元一世纪，凯尔特人的一支盖尔人登陆爱尔兰岛并定居，形成了许多部落邦国。

1.2 罗马帝国晚期基督教传入爱尔兰，为盖尔人诸邦国所信奉。

1.3 公元8世纪后期，维京海盗劫掠爱尔兰，一部分定居，并建立了一批城镇。

2、英国人来了

公元1170年，英格兰伯爵彭布鲁克应邀在爱尔兰巴金本登陆，后成为伦斯特邦国的邦王。英国人开始介入爱尔兰历史。

二、英国征服与爱尔兰大饥荒

1、英国国王成为爱尔兰国王

1.1 英格兰贵族纷纷来到爱尔兰，占据土地， 成为领主。

1.2 爱尔兰领主逐渐抛弃英国生活方式，引起英国国王警惕。

1.3 1534年，英国国王亨利八世要求爱尔兰领主把土地交出来，爱尔兰全部土地的所有权转到英国国王手上。

1.4 伊丽莎白女王镇压爱尔兰领主的反抗，确立英国对爱尔兰的统治。

2、英国的殖民化政策

2 .1 爱尔兰人信奉天主教，英国人信奉基督新教，因此爱尔兰人把英国对爱尔兰的殖民视作新教徒对天主教徒的迫害。

2.2 英国向爱尔兰殖民，并在其北部建立起了新教徒聚居区。与此同时，英国政府极力打压天主教徒。

2.3 爱尔兰沦为英国边缘地区，并错过了全球化和工业革命。

3、爱尔兰大饥荒

3.1 马铃薯绝收，让以吃马铃薯为生的爱尔兰人食物匮乏，成批饿死。

3.2 英国政府冷漠以对，加重饥荒。

3.3 从1845年到1849年，爱尔兰大饥荒导致100万人饿死，另有150万人移居海外。

三、为独立而斗争

1、武装斗争

1.1 大饥荒期间爱尔兰各地不断有暴动，但没有对英国统治构成威胁。

1.2 詹姆斯·斯蒂芬斯创建爱尔兰兄弟会，拿起武器投入战斗，将爱尔兰建设成要将独立的民主共和国。

1.3 爱尔兰兄弟会于1867年发动起义，但失败。

2、政治斗争

1.1 一部分爱尔兰议员组成爱尔兰土地同盟，要求爱尔兰人拿回土地所有权。

1.2 英国议会于1903年通过《土地购置法》，授予爱尔兰佃农土地所有权。

1.3 知识分子掀起爱尔兰文艺复兴运动，突出爱尔兰民族的文化特性。

1.4 1905年，新芬党成立，致力于推动爱尔兰自治。

3、自治许诺与复活节起义

1.1 第一次世界大战爆发，为安抚民众，英国许诺战后给予爱尔兰自治地位，条件是爱尔兰人帮英国打仗。爱尔兰人纷纷参军。

1.2 激进派认为这是阴谋，于1916年复活节在都柏林举行起义，惨遭英军镇压。

四、从内战到和解

1、自治与内战

1.1 1921年爱尔兰与英国签订《英爱条约》，爱尔兰获得自治领地位，叫做爱尔兰自由邦。为英联邦成员，但享有内政和外交全权，并拥有军队。

1.2 爱尔兰北部六个郡仍然归英国管辖。多数爱尔兰人接受现状，但新芬党中的共和派反对，由此产生了“北爱问题”。

1.3 共和派脱离新芬党，组织爱尔兰共和军试图统一整个爱尔兰。作为执政党的新芬党进行镇压。内战爆发。

1.4 政府军获胜，爱尔兰共和军被迫转入地下搞恐怖活动。

2、愈演愈烈的恐怖活动

2.1 1948年爱尔兰正式独立，但爱尔兰共和军不依不饶。

2.2 爱尔兰的乡村和爱尔兰人的反英情绪，是爱尔兰共和军的生存土壤。

2.3 爱尔兰共和军在北爱地区和英国本土大搞恐怖活动，从1970年到1994年，总计导致超过3100人死亡。该组织被很多国家定性为恐怖组织。

3、走向和解

3.1 1993年12月，英爱两国首相发表《唐宁街宣言》，双方关系缓和。

3.2 在时任美国总统克林顿的调解下，爱尔兰共和军宣布暂停使用武力。

3.3 1998年英爱签署《友好星期五协议》，爱尔兰共和军宣布放弃无力。英爱实现和解。

4、经济奇迹

4.1 和平稳定的国内环境促使外资涌入投资，促进了爱尔兰经济。同时爱尔兰将重心放在软件业上，成为仅次于印度的软件出口大国。

4.2 爱尔兰经济腾飞，至2003年年人均GDP超过欧盟平均水平的1.36倍。

4.3 爱尔兰民生大幅度提高，以超过9万美元的人均GDP位居全球第二，其生活水平已经能同高福利国家比肩。
