1428.历史丨《另一个纽伦堡》：正义的审判，送日本甲级战犯上绞刑架

发表于 2021-09-03 分类于 听书笔记

正义的审判，送日本甲级战犯上绞刑架

精华笔记

一、**前期准备：东京审判的法理依据和国际法庭成立**

1、法理依据

1.1 1945年7月，中国、美国和英国发布《波茨坦公告》，敦促日本无条件投降。后苏联加入。8月14日宣布接受《波茨坦公告》，向盟军无条件投降。

1.2 《波茨坦公告》第十条规定对日本战犯，同盟国将予以审判。这构成了战后审判最重要的法理依据。

2、审判的准备工作

2.1 制定战犯名单，实施抓捕。逮捕上千名日本战犯，被分为甲级、乙级、丙级三个等级。先行审判乙级和丙级战犯。

2.2 建立国际法庭，公开28名甲级战犯审判。1946年1月，由十一国代表组成的远东国际军事法庭正式成立。

2.3 检方和辩方做准备。

2.3.1 日本人利用时间差销毁证据。

2.3.2 检察官克服重重困难，搜集了大量证据，给每一名甲级战犯建立了交叉档案。

2.3.3 28名甲级战犯都指派了辩护律师。

二、**策略与交锋：控辩双方的法庭表现**

1、检方宣读起诉书，辩方避实击虚。

1.1 首席检察官约瑟夫·季南宣读针对28名甲级战犯的起诉书，战犯被控以共三组罪名：一，破坏和平罪及共谋罪；二，谋杀罪；三，危害人类罪。

1.2 辩方律师向法庭发起挑战，以首席法官威廉·韦伯曾参与调查日军暴行为由，要求其回避。法庭驳回。

1.3 辩方律师要求更换所有现任法官，代之以中立国法官，保证审判公正性。法庭再度驳回。

2、控方举证、辩方律师反驳，双方正面交锋。

2.1 进入举证环节，检方出示大量证据，包括南京大屠杀、马尼拉大屠杀等，证实日军暴行。

2.2 辩方律师采取承认大屠杀，但极力撇清被告和屠杀存在关系的辩护策略。检方则举出相应证据，证明被告知道日军暴行的存在，并亲自下令施暴。

辩方律师进行调整，将辩护策略从“不知情”变成“不得已”，遭到检方有力驳斥。

3、被告上台作证，辩方全线崩盘。

3.1 被告上台后颐指气使、蛮横无理，充分暴露了侵略者嘴脸。

3.2 在检方的盘问下被告全线崩溃。

三、**宣判与影响：东京审判的历史意义。**

1、宣判与执行

1948年11月12日，远东国际军事法庭最后一次开庭。25名被告均被判有罪，其中东条英机、松井石根、板垣征四郎等6人被处以绞刑，并于12月23日零点执行。

2、缺憾

2.1 没有审判天皇。

2.2 没有追究日军细菌战的责任。

2.3 庭审过程拖得太长。

3、意义

3.1 将国际法从纸面落到实处。

3.2 维护和平秩序，警醒日本民众。
