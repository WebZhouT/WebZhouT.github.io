1386.历史丨《剑桥现代印度史（上）》：殖民遗产+种姓制+家族统治+民粹主义＝现代印度？

发表于 2021-07-08 分类于 听书笔记

殖民遗产+种姓制+家族统治+民粹主义＝现代印度？

精华笔记

一、敌从海上来：东印度公司敲开印度大门

1、成立东印度公司

1.1 1600年，英国女王伊丽莎白授权富商创办。

1.2 股份制企业，拥有垄断东方贸易特权。

1.3 早期目标东南亚，但已被荷兰捷足先登，改业务至印度。

2、印度地缘政治

2.1 一面背山，三面环海，威胁主要来自北方。

2.2中北部为平原易攻难守，入侵者多穿越开伯尔山口进入印度，创建王朝。

2.3 印度统治者思维定式，紧盯北部边疆，轻视防范海洋。

3、占据海岸线

3.1 沿印度海岸线建立贸易据点。

3.2 通过与莫卧儿帝国交易，购买土邦王公管理的海岸线非核心区域。

3.2 至18世纪，东印度公司已建立三大贸易据点：孟买、马德拉斯和孟加拉，几乎囊括印度海岸线。

二、全面殖民：一家公司征服一个帝国

1、击败孟加拉

1.1 孟加拉盛产棉花和生丝，是英国棉纺织业最重要的原材料基地。

1.2 东印度公司在此建立工厂、庄园和军营，当地王室收入被压榨，发动突击。

1.3 英军以微弱代价取胜，孟加拉王公被处死。

14 扶植傀儡政权，获得巨额赔款，成为实际统治者，并向周边扩张。

2、设立印度总督

2.1 1764年，莫卧儿皇帝发动布克萨尔战役，莫卧儿帝国沦为东印度公司附庸。

2.2 1772年，东印度公司设立英属印度总督，开始全面征服印度。

2.3 东印度公司由贸易企业，转变为殖民政府。

3、维系殖民统治

3.1 垄断行政、军事关键职位，操控印度。

3.2 进行残酷经济掠夺，确保收入稳定。

3.3 以印度为基点殖民扩张，发动鸦片战争，殖民亚洲。

三、殖民体制升级：从民族大起义到代理人统治

1、民族大起义

1.1 1857年，东印度公司强迫印军使用涂有牛油和猪油的弹药包，引发印度士兵反抗。

1.2 起义军攻入莫卧儿首都德里，号召全民反抗殖民统治。

1.3 印度各地相继兵变，发起印度民族大起义，英国失去印度中北部控制。

1.4 英政府集军队，攻陷德里，莫卧儿帝国正式覆灭。

2、代理人统治

2.1 殖民者拉拢部分印度人成为统治代理人。

2.2 通过分享经济发展权利、提供高级行政职位，以及培养文化上亲英人士，与高种姓子弟和殖民者在利益、精神、情感上深度捆绑。

2.3 代理人统治强化了种姓制，也萌生出强烈的民族意识，精英群体开始谋求印度独立。
