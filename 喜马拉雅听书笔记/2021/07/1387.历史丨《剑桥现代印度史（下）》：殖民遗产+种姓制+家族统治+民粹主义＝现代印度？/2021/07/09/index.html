1387.历史丨《剑桥现代印度史（下）》：殖民遗产+种姓制+家族统治+民粹主义＝现代印度？

发表于 2021-07-09 分类于 听书笔记

> 殖民遗产+种姓制+家族统治+民粹主义＝现代印度？

精华笔记

一、走向独立：尼赫鲁、甘地与非暴力不合作

1、尼赫鲁

1.1 印度开国总理，高种姓富家子弟，本土精英。

1.2 在英留学期间觉醒，决定改变现状转向民族主义。

1.3 加入国大党，为印度人争取参政权，呼吁保护本土工商业。

2、压制民族主义

2.1 印度当局挑唆印度教徒和穆斯林矛盾，压制民族主义，国大党发动民众反抗。

2.2 英政府拉拢本地精英，国大党接受孟加拉被分割事实，反抗活动减弱。

2.3 一战爆发，印度为英参战死伤惨烈，民众示威要求印度自治，民族主义再度兴起。

2.4 国大党组织罢工、罢课、恐怖活动，难形成有组织群众运动，向英施压有限。

3、甘地与非暴力不合作

3.1 1920年发起第一次非暴力不合作运动，不买英国货，只买印度商品，获民众广泛支持。

3.2 国大党全力支持，趁机建立基层党组织，从精英政党转向全民政党。

3.3 1935年《印度政府法》颁布，开放选举，印度实现省级自治。

3.4 二战爆发英国无力维护海外殖民地，印度独立。

二、尼赫鲁家族统治：正面作用和负面遗产

1、正面作用

1.1确立基本政治秩序，初步建立起工业体系。

1.2 赋予民族平等政治权利，消除民族隔阂和阶层差异，社会步入稳定期。

1.3 在经济上发展“尼赫鲁社会主义”，大力发展重工业，扶持计算机产业，为软件大国奠定基础。

2、负面遗产

2.1 称霸周边，边界纠纷不断，挑起中印边境战争，国际形象一落千丈。

2.2 贫富差距拉大，社会两极分化。

六、1990年代以来：人民党崛起和印度前景

1、人民党崛起

1.1 1991年，国大党继任者遇刺，影响力下降，印度人民党成国会第一大党。

1.2 在政治上分享权利，开放中高级职位，吸引低种姓人群，获得支持。

1.3 吸引海外投资，扶植本土产业，同低种姓群体分享经济红利。

1.4 复兴印度教，塑造国家认同感，引穆斯林反感和不安，地区暴力冲突不断。

2、印度前景

2.2 仍需解决政治结构、贫困、宗教和民族矛盾等方面问题。

2.3 普及教育，促进公平公正发展机会。
