1351.历史丨《西班牙何以成为西班牙（上）》：比英国还早的“日不落帝国”，从鼎盛到衰落

发表于 2021-05-20 分类于 听书笔记

比英国还早的“日不落帝国”，从鼎盛到衰落

精华笔记

一、各方势力角逐的战场

1.迦太基人VS罗马人

1.1 公元前264年到前241年，罗马与迦太基展开攻防战。罗马人拿下西西里岛和撒丁岛，直指北非。

1.2 迦太基名将汉尼拔在西班牙东南部登陆，以此为根据地，杀入罗马本土。罗马人措手不及。

1.3 公元前211年，罗马突袭西班牙。迦太基被迫将它在西班牙的领地割让给罗马。

2.罗马化

2.1 罗马人在西班牙建造城市、修筑道路，促进贸易往来和文化交流，将不同区域连接起来。

2.2 罗马人还大力开发自然资源，特别是金、银、铜等贵金属。

2.3 塑造了西班牙文化，传播基督教。

3.西哥特人VS摩尔人VS基督教王国

3.1 公元4世纪，罗马帝国崩溃，西哥特人随后翻越比利牛斯山，建立西哥特王国。他们皈依了基督教。

3.2 公元711年，摩尔人在西班牙最南端的直布罗陀登陆。西哥特国王被杀，摩尔人占据西班牙南部和东部，建立科尔多瓦王朝。

3.3 一部分基督徒转移到西班牙北部和西部，形成基督教各王国对抗科尔多瓦王朝的格局。

二、收复失地运动

1.拉锯与反超

1.1 摩尔人同基督教各王国在马德里一线反复拉锯。总体而言，摩尔人占优势。

1.2 10世纪初科尔多瓦王朝达到鼎盛，北方的那些基督教王国，实力较弱。

1.3 摩尔人因分裂和内斗不断衰落，基督教各王国的实力逐渐反超，发起收复失地运动。

  1. 西班牙王国的诞生

2.1 公元1085年，国王阿方索六世夺得马德里和托莱多，自称“全西班牙人的皇帝”，正式打出了“收复失地运动”的旗号。

2.2 收复失地运动历经400多年才大功告成，原因有三个：

2.2.1 摩尔人顽强抵抗。

2.2.2 西班牙人内外纷争不断。

2.2.3 黑死病肆虐。

2.3 基督教各王国完成整合，建立西班牙王国。1492年完成统一。

三、建立全球性帝国

  1. 划分全球势力范围

1.1 哥伦布远航拉开地理大发现的帷幕，成为欧洲人殖民美洲的先声。

1.2 西班牙和葡萄牙竞争激烈，双方以子午线为界，约束各自的行动。子午线以西的新发现地区，归西班牙，以东则归葡萄牙。它开了欧洲列强划分全球势力范围的先河。

  1. 征服美洲

2.1 1519年，埃尔南·科尔特斯在今墨西哥登陆，摧毁阿兹特克帝国。

2.2 1532年，弗朗西斯科·皮萨罗在今秘鲁登陆，摧毁印加帝国。

2.3 17世纪中叶，西班牙殖民地从今美国南部，穿越墨西哥、古巴、危地马拉，延伸到南美洲的秘鲁、智利、阿根廷。

3.在欧洲的扩张

3.1 西班牙王室同哈布斯堡家族联姻。将今德国、奥地利、法国东部、荷兰、比利时等地区收入囊中。

3.2 成为世界上第一个“日不落帝国”缩影。
