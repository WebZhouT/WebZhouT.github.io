1357.历史丨《当权的第三帝国（下）》：自私与冷漠，将普通人推入纳粹挖好的坑

发表于 2021-05-28 分类于 听书笔记

自私与冷漠，将普通人推入纳粹挖好的坑

精华笔记

一、为巩固统治，纳粹用了哪些强硬手段

1、宣誓效忠

1.1 要求军人宣誓，无条件效忠希特勒个人。

1.2 进一步要求、官员、公务员等都宣誓效忠希特勒个人。

2、控制地方行政系统

废除地方选举，关闭州政府，由纳粹党委派的州总督进行直接管理。

3、迫害一切心怀不满的人

3.1 设立人民法庭，惩处政治犯。

3.2 建立特务系统——盖世太保，监视民众，并鼓励民众举报。

3.3 设立集中营，凡是不与纳粹合作的，一律关押。

二、纳粹统治的柔性手段

1、宣传洗脑

1.1 取缔限制非纳粹媒体。

1.2 市面上只有纳粹的官方报纸和广播，人民只听得到一种声音——纳粹希望他们听到的声音。

1.3 通过《意志的胜利》这样的电影，对人民进行洗脑。

2、用娱乐麻痹人民

2.1 鼓励民众观看喜剧片、剧情片，阅读流行小说，远离严肃思考。

2.2 针对劳工阶级，推出大众旅游服务，缓解他们的不满情绪。

3、塑造青少年

纳粹要求10到18岁的孩子参加希特勒青年团。接受系统的法西斯教育，读纳粹编撰的教材、唱纳粹歌曲，还要进行军事训练，为未来的战争做准备。

三、德国人是抱着怎样的心态走向战争的

1、纳粹的洗脑效果是有限的

1.1 统治时间太短，渗透力不足。

1.2 法西斯主义东拼西凑，本身缺乏吸引力。

1.3 民众的真实处境，同宣传严重不符合。

2、德国人在冷漠麻木中一步步陷入战争

2.1 到1930年代后期，相当一部分德国人对纳粹统治已非常冷漠。

2.2 纳粹相继吞并了奥地利、捷克斯洛伐克等，而英法采取绥靖政策，步步退让，这使德国人以为战争永远不会降临。

2.3 1939年9月1日，德军入侵波兰，第二次世界大战正式爆发。至此，德国人追悔莫及。
