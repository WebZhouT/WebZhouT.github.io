1361.历史丨《低地国家史（上）》：全世界最富裕经济体是怎样锤炼成就的

发表于 2021-06-03 分类于 听书笔记

小到买件衣服，大到国家大事，生活中处处是谈判

精华笔记

全世界最富裕经济体是怎样锤炼成就的

低地国家是对荷兰、比利时和卢森堡的统称。它们位于欧洲西北部，地势平坦，海拔最低处比海平面略低，所以称作尼德兰，也就是“低洼之地”的意思。

一、从罗马边境到法兰克核心区

1、罗马帝国的东部边境

1.1 公元前600年，高卢人的分支比利其高卢人开始统治低地国家。

1.2 罗马名将恺撒征服高卢，罗马帝国创始人奥古斯都设立比利时高卢，囊括今法国东北部、卢森堡、比利时、荷兰，以及现属德国的莱茵兰。比利时高卢成为罗马帝国的东部边境。

1.3 比利时高卢罗马化。

2、法兰克帝国的核心区

2.1 公元476年西罗马帝国灭亡。481年法兰克部落首领克洛维，以低地国家和今法国北部为基本盘，创建法兰克王国。

2.2 查理曼当政时，法兰克王国变身为法兰克帝国。首都定在今德国、比利时与荷兰的交汇处亚琛。低地国家升级成帝国核心区。

2.3 查理曼驾崩后法兰克帝国形成东、西法兰克王国。卢森堡和荷兰归东法兰克王国，即德意志王国；比利时分为两部分：东部归德意志，西部归西法兰克王国，即法兰西王国。

二、从边缘地带到文明的十字路口

1、高度自治的封建主

1.1 中世纪的法兰西和德意志内部分裂，中央政府权力有限，低地国家由大大小小的封建主组成，拥有很大的自治权。

1.2低地国家的封建主建造城堡和教堂，征收通行费和税款，开发矿产、森林、水资源，并且铸造钱币、征募军队。同时还能影响朝政，如佛兰德斯伯爵国。

2、商业网络与文化艺术

2.1 相似的地理环境与生活习俗使封建主之间的联系十分紧密，编织成一张覆盖欧洲的商业网络，安特卫普、布鲁塞尔、根特、鹿特丹等成为北欧商贸中心。

2.2 各种商品、人才汇聚于此，促进文化繁荣，产生了凡·艾克这样的艺术家，开启了欧洲油画艺术。

三、早期全球化的受益者

1、哈布斯堡家族的统治

1.1 发迹于奥地利的哈布斯堡家族经过一系列政治联姻，据有今德国、西班牙、捷克等在内的广阔地区。低地国家同样收入囊中。

1.2 从16世纪起，西班牙在美洲建立殖民地，并从那里掠夺白银、皮毛、香料等财富运往低地国家，再传入欧洲各地。低地国家由此被纳入早期全球化中。

2、反抗哈布斯堡家族

2.1 哈布斯堡家族保守、严厉的宗教，和沉重的税收政策，令低地国家人民不满。

2.2 1566年爆发“尼德兰革命”，反抗哈布斯堡统治。

2.3 1581年，低地国家北方的七个省和南方的十个省，宣布成立尼德兰联省共和国。南方十省后来退出，北方七个省成为“荷兰共和国”。

3、荷兰共和国的兴盛

3.1 荷兰向海外殖民，在北美和东南亚都建立了殖民地。

3.2 大力发展航海业，保险业也应运而生。银行贷款、票据贴现、股票和期货交易等新兴金融业态蓬勃发展。阿姆斯特丹成为国际金融中心。

3.3 文化与科学发达，出现了画家鲁本斯、伦勃朗，哲学家笛卡尔、斯宾诺莎，微生物学创始人列文虎克等。
