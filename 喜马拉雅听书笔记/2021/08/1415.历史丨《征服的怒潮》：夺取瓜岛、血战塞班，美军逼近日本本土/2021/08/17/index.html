1415.历史丨《征服的怒潮》：夺取瓜岛、血战塞班，美军逼近日本本土

发表于 2021-08-17 分类于 听书笔记

夺取瓜岛、血战塞班，美军逼近日本本土

精华笔记

一、战略调整：战争态势与盟军内部协调

1、向欧洲倾斜

1.1 中途岛战役后，日军进攻势头得到遏制，英国和苏联敦促美国实行“欧洲优先”战略。

1.2 同盟国经过协商，决定欧洲获得全部军事资源的70%，太平洋战场获得30%。

1.3 有限的军事资源决定了，美军在太平洋战场只能选择特定区域作为主攻方向。

2、美军确定反攻方向

2.1 日军将拉包尔和瓜岛、图拉吉岛连成一条战线，试图以此作为入侵澳大利亚的跳板。

2.2 美军决定攻占瓜岛，在日军战线上敲开一个洞。

2.3 弗莱彻中将率领由三个航母编队组成的特混舰队，为攻占瓜岛提供支持。登陆作战则交给了海军陆战队第一师。

二、争夺瓜岛：美军打破日军防线

1、开局

1.1 1942年8月7日，美陆战一师在瓜岛登陆，没有遇到抵抗。

1.2 日军战机从1000多公里外的拉包尔赶赴瓜岛驰援，被美军击退。

1.3 8月8日晚，日军第八舰队在瓜岛北部的萨沃湾击沉美军4艘巡洋舰、重创一艘，还击伤2艘驱逐舰，削弱了美军制海权。

2、僵局

2.1 日军尝试在瓜岛登陆，袭击美军，但失败。

2.2 8月24日，日本海军重创美军两艘重型航母，削弱美军的制海权和制空权。日军利用机会向瓜岛大规模增兵，使兵力超过陆战一师。

2.3 日军不断增兵，并不断轰炸瓜岛美军。美军做出重大调整：将南太平洋战区司令戈姆利撤职，命哈尔西接任。

3、破局

3.1 哈尔西积极求战，日军也派出4艘航母、4艘战列舰，总计61艘军舰起航，争夺瓜岛制海权。

3.2 10月26日，双方航母对决，日军三艘航母重创，美军一艘重创、一艘沉没。日军航空兵大量损耗，已无力夺得瓜岛制空权。

3.3瓜岛上的36000名日军因海空补给被切断，成批饿死。1943年2月，日军从瓜岛撤出约一万名士兵，其余人葬身瓜岛。瓜岛争夺战以美军胜利告终。

三、拖垮对手：蛙跳战术和消耗战术

1、蛙跳战术

1.1 美军伏击联合舰队总司令山本五十六，加强攻占拉包尔的信心。

1.2 美军攻占拉包尔东面的蒙达岛，但1200人阵亡。而拉包尔有十万守军，势必付出更大代价。美军决定调整战术。

1.3 美国海军作战部长欧内斯特·金下令：绕开拉包尔。美军战术出现重大变化。此后凡是日军重点防守区域，美军就绕开，转而攻击其防线上的薄弱环节。这就是“蛙跳战术”。

1.4 美军用蛙跳战术向东南亚逼近。

2、消耗战术

2.1 为确保东南亚的资源运回日本，日军花了一条绝对国防圈，将南太平洋的新几内亚西部，西太平洋的印尼群岛、马来半岛、菲律宾，以及中太平洋的关岛、塞班岛悉数纳入。

2.2 美军掐准日军的命门，重点打击其海运线，用消耗战拖垮日本。

2.3 潜艇发挥作用。太平洋战争期间，美军潜艇共击沉201艘日本军舰、1100多艘商船。对日本的海线造成了毁灭性打击。

2.4 日军跟着美军节奏，大打消耗战，将所剩不多的航空兵无止尽地在拉包尔上空消耗。

3、美军不断挺近

1943年底，美军攻占日本联合舰队的锚地塔拉瓦环礁，下一个目标是中太平洋最重要的据点——关岛和塞班岛。

四、逼近本土：血战塞班和菲律宾海海战

1、登陆塞班岛

1.1 1944年6月11日起美军进行火力准备，摧毁了塞班岛上大部分地面工事。15日登陆。

1.2 岛上日军顽强抵抗，首日美军即伤亡5500人。战况胶着。

2、菲律宾海海战

2.1 美日海军争夺塞班岛制海权，战场选在菲律宾海。

2.2 6月19日上午，美日海军进行航母对决。结果日军损失三艘重型航母、400多架飞机，日美战损比高达12：1。

2.3 菲律宾海海战以美军获得压倒性胜利告终，这标志着美军完全掌握了太平洋的制海权。

3、解放塞班和关岛

3.1 美军切断日军海空补给，稳步推进。日军缺粮少药，战斗力大幅度下降。

3.2 日军将领斋藤义次和南云忠一相继自杀。7月7日凌晨，残余日军发动自杀式攻击，被美军全歼。

3.3 7月9日，美军攻克塞班岛最后一处日军阵地，宣布塞班解放。8月12日，美军夺关岛。

4、此战意义

4.1 塞班战役与诺曼底战役几乎同步展开和结束，标志着美国的军工业潜能已完全激活，美军具备在欧洲和太平洋这两大战场上同时开展大规模作战的能力。

4.2 对日本来说，丢失塞班岛和关岛，东京和关东工业区已处于美军轰战机的航程以内，灭顶之灾就要降临。民众也对战争前景产生动摇。
