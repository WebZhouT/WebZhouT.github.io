历史丨《燃烧的大洋》：从全线崩溃到绝地反击，美军是怎样反超日军的

发表于 2021-08-16 分类于 听书笔记

> 从全线崩溃到绝地反击，美军是怎样反超日军的

精华笔记

一、战云密布：日本军国主义的扩张

1、日本扩张与英美制约

1.1 明治维新以来，日本成为唯一能同西方国家抗衡的亚洲国家。这激发了它称霸东亚乃至整个亚洲的野心。

1.2 第一世界大战中日本向德国宣战，一战后，日本获得了德国在太平洋上的几块殖民地，将势力渗透到太平洋西部、南部和中部。对英美等国在东南亚的利益构成威胁。

1.3 华盛顿条约规定，英、美、日三国主力舰总吨位的比例为5：5：3。制约日本。

2、日本走向军国主义

2.1 受条约影响，日本海军缩减规模，引起强烈不满。仰赖海军为生的造船业也大受影响。

2.2 在军方力推下，日本舆论猛烈抨击《华盛顿条约》。军国主义甚嚣尘上。

2.3 1937年日本挑起“七·七事变”，中日战争全面爆发。日本独霸中国的企图，严重损害英美利益。

3、日本决意对英美开战

3.1 美国对日本实施经济制裁和武器、石油禁运，捏住了日本军工业生产的命门。

3.2 日本军方提出南进政策，即攻占东南亚，获得石油、橡胶、金属等军事物资。

二、偷袭珍珠港：日本激怒美国

1、山本五十六力排众议，锁定珍珠港

1.1 日本海军最高指挥机关军令部的主张，开战后任由美军舰队跨越太平洋，待其驶入日本海域，联合舰队凭借战列舰将对手一举消灭。

1.2 联合舰队司令山本五十六认为此举无异于慢性自杀，他认为必须速战速决，短期内使美国太平洋舰队丧失战斗力，日本才有赢面。

1.3 山本五十六决意偷袭美国太平洋舰队总部——珍珠港。

2、战前准备

2.1 解决技术问题，使鱼雷能够适应珍珠港的水文条件。

2.2 麻痹美国，保证袭击的突然性。

2.3 海军官兵取消休息日，加紧训练。

3、偷袭珍珠港

3.1 包括6艘航母在内的23艘军舰组成第一航空舰队，由南云忠一指挥，秘密潜伏至珍珠港附近。

3.2 1941年12月7日上午，351架日军战机分两批次奇袭珍珠港，击毁美军战机188架，炸沉战列舰4艘、重创3艘，击伤巡洋舰和驱逐舰9艘，美军阵亡2400多人。

3.3 美军航母因出海执行任务，躲过一劫。

4、美国对日宣战

1941年12月8日，美国对日宣战，正式加入第二次世界大战。

三、全线崩溃：日军席卷东南亚

1、日军夺取制海权和制空权

1.1 偷袭珍珠港的同时，日军飞机轰炸了美军位于菲律宾，英军位于马来半岛、缅甸和中国香港的军事基地。仅仅三天，盟军在东南亚的空中力量被摧毁殆尽。日军夺得制空权。

1.2 日军击沉英军在东南亚的主力战舰——反击号和威尔士亲王号，夺得制海权。

2、日军发起全面攻击

2.1 日军从菲律宾北部登陆，逼近菲律宾首府马尼拉。美菲联军节节败退。1942年4月，弹尽粮绝的美菲联军投降。菲律宾沦陷。

2.2 与此同时，日军向英属马来亚，即今天的马来西亚和新加坡发起进攻。1942年2月，英军在马来半岛的最后据点——新加坡，向日军投降。

2.3 日军兵锋直指荷属东印度，即今天的印度尼西亚。至1942年3月，荷属东印度沦陷。

2.4 3月，日军占领新不列颠岛，逼近澳大利亚东北部海域。澳洲军事基地达尔文港遭多轮空袭，死伤惨重。

四、惊魂初定：美军调整部署、总结经验

1、调整部署

1.1 美国太平洋舰队总司令金梅尔因玩忽职守被撤职，罗斯福总统钦点切斯特·尼米兹接任。

1.2 任命海军上将欧内斯特·金为海军作战部长兼美国舰队总司令，负责战争规划、行政管理、作战指挥。

1.3 尼米兹负责重建太平洋舰队，鼓舞士气；欧内斯特·金在尊奉“欧洲优先”的原则下，尽全力支持太平洋战场，调集了大量军事资源。

2、总结经验

2.1 珍珠港事件证明“战列舰决战”的思维已落伍，航母才是海战主力。美国海军改弦更张，叫停了建设中的战列舰，全力打造航空母舰。

2.2 鉴于美军主战飞机性能落后于日本零式战斗机，美军一方面改善战机性能，一方面摸索出一套对付零式战斗机的方法。

五、山姆大叔的复仇：从轰炸东京到决战中途岛

1、轰炸东京

1.1 罗斯福总统指示轰炸东京，以提振美国人的士气。

1.2 尼米兹拿出两艘航母，由哈尔西海军中将率领，向日本海域推进。

1.3 1942年4月18日，舰队距离东京约1000公里，16架B-25远程轰炸机顶着狂风暴雨起飞，轰炸东京。

1.4 轰炸东京本身造成的损失微不足道，但对日本人的心理冲击很大。山本五十六决定消灭美军航母。

2、珊瑚海海战

2.1 日军计划夺取澳大利亚东北部门户——珊瑚海的制海权，随后登陆澳洲，铲除美军基地，杜绝美军再次轰炸东京的可能性。

2.2 尼米兹命弗莱彻少将指挥的第17特混舰队，下辖两艘航母，出海迎敌。

2.3 珊瑚海海战日军失去一艘轻型航母，一艘遭重创，美军两艘重型航母一沉一伤。日军略占上风。但日军没能占领澳大利亚，战略上失败。

3、中途岛战役

3.1 山本五十六决意攻占中途岛，

作为进攻夏威夷的前进基地。1942年5月，南云忠一率领第一航空舰队从广岛出发，尼米兹命弗莱彻少将率三艘航母迎战。

3.2 三回合较量

3.2.1 日军空袭中途岛美军基地，美军随即反击，但没有击中一艘日军航母。

3.2.2 南云忠一在再次轰炸美军基地还是轰炸美军航母之间犹豫不决，浪费了大量时间。关键时刻，美军33架俯冲轰炸机飞临第一航空舰队上空，接连击沉日军三艘重型航母。

3.2.3 日军仅存的一艘航母飞龙号反击，击沉美军航母约克城号。但飞龙号也被美军击沉。

3.3 中途岛战役美军损失一艘航母，击沉日军四艘重型航母，取得开战以来最大胜利。日军丧失了发动大规模攻击的信心，美军将逐渐掌握太平洋战场的主导权。
