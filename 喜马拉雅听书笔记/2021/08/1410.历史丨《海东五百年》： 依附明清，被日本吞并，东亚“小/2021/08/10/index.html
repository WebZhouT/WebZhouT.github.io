> 依附明清，被日本吞并，东亚“小中华”的兴衰荣辱

精华笔记

一、王朝初建：东亚变局与李氏朝鲜的开创

1、在明朝和北元之间

1.1 1368年朱元璋建立明朝，明军北伐，元顺帝仓皇北逃，重返蒙古高原，称北元政权。

1.2 高丽表示向明朝朝贡，成为明朝藩属国。

1.3 高丽国王采取两面手法，既接受明朝册封，又接受北元册封，令朱元璋不满。

2、李成桂建立朝鲜王朝

2.1 高丽国王试图占据辽东地区，命高丽名将李成桂领军北伐。

2.2 李成桂率军反戈一击，以“清君侧”名义掌握朝政。

2.3 1392年李成桂登上国王宝座，并为新王朝定名“朝鲜”。明太祖朱元璋认可。

3、朝鲜与明朝关系强化

3.1 1400年李成桂之子李芳远发动政变，掌握朝鲜国政。同时期朱元璋之子朱棣发动靖难之役，李芳远暗中支持。

3.2 朱棣推翻建文帝，自立为帝。为报答李芳远，册封其为朝鲜国王。

3.3 明朝与朝鲜正式确立宗藩关系，朝鲜尊明朝为宗主国，定期朝贡；明朝一般不干涉其内政。

二、移风易俗：朝鲜的儒教化

1、看中儒家的忠君思想

1.1 高丽王朝时期朝鲜即效仿中国，建立科举制度，并涌现出郑梦周这样的大儒。

1.2 郑梦周因反对李成桂造反而被杀，但他倡导的“忠君”思想李氏朝鲜大肆宣扬，并将郑梦周入祀孔庙。

2、世宗进一步强化儒教

2.1 完善礼法，打击佛教

2.2 刊印典籍、创制谚文，向民众宣教

3、推广儒教的效果

3.1 民众生活发生很大变化，按照儒教学说行为处事，女性地位下降，从拥有相对独立性到依附于男性。

3.2 两班士大夫执着于嫡子继承制，导致社会流动性降低，阶层固化。

3.3 朝鲜国王的权力受到两班贵族制约，而国王为了扩权，挑唆两班内斗，导致党争不断。

三、从壬辰倭乱到归附清朝

1、日本入侵朝鲜

1.1 16世纪丰臣秀吉基本统一日本，谋划攻占朝鲜，并侵略中国。

1.2 1592年农历四月十三日，日军在朝鲜登陆，战争打响。史称“壬辰倭乱”。

1.3 朝鲜军队节节败退，汉城、平壤尽皆沦陷。朝鲜国王向明朝求救。

2、明朝出兵朝鲜

2.1 明万历帝命20万明军入朝救援，接连收复平壤、汉城。

2.2 明朝与日本谈判，朝鲜内部党争不断，导致战争绵延七年。

2.3 1598年丰臣秀吉病死，日军全线撤离，壬辰倭乱结束。

2.4 战争影响：消耗了明朝大量人力、物力和财力，以至于它对边疆地区的控制力大幅度削弱。给后金在辽东的崛起创造了空间。

3、朝鲜归附清朝

3.1 1636年后金大汗皇太极称帝，改国号为“大清”。皇太极要求朝鲜臣服，遭到拒绝。于是清军纵马南下，直扑汉城。

3.2 朝鲜君臣逃入南汉山城，被清军围困。在内外交困的情况下，朝鲜国王被迫臣服。

3.3 1644年清军入关，清朝与朝鲜的宗藩关系已不可动摇。

四、走向衰亡：近代朝鲜的悲剧命运

1、朝鲜对清朝观感的转变

1.1 朝鲜君臣从“夷夏之辨”的角度，视清朝为蛮夷之邦，自居“小中华”。

1.2 康熙帝平定三藩之乱，加上对朝鲜采取怀柔政策，朝鲜君臣的观感大为转变。

1.3 作为藩属国，朝鲜纳入了清朝的朝贡贸易体系，收益颇丰。

2、国际环境巨变

2.1 19世纪中叶，法国、美国等相继要求朝鲜开国，遭朝鲜君臣拒绝。

2.2 日本自明治维新后国力大增，“征韩论”甚嚣尘上。

2.3 朝鲜宫斗不断，国力进一步衰落，无力抵御列强进逼。

3、朝鲜被日本吞并

3.1 日本逼迫朝鲜签订条约，开放通商口岸，并赋予日本人治外法权。

3.2 朝鲜发生兵变，受高宗之邀，清军入朝平乱。日本也以保护侨民为借口出兵。事后日本迫使朝鲜签订条约，获得驻军权。

3.3 1894年中日甲午战争爆发，日军取胜，从而将清朝彻底驱逐出朝鲜半岛。

3.4 1910年，日本吞并朝鲜，朝鲜王国灭亡。
